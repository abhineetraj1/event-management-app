# Event Management App

<img src="https://raw.githubusercontent.com/abhineetraj1/event-management-app/main/screenshot.png" width="100%">

The Event Management App is a Python Tkinter application that allows users to store and manage customer travel information. It provides features to add, search, and delete travel information. The app offers a dark mode theme with a black and silver color scheme.

## Features

- Add Travel Information: Users can enter details such as the place of visit, home place, email, phone number, and customer name to store customer travel information.

- Search Travel Information: Users can search for customer travel information by entering a keyword. The app will display matching records based on the keyword.

- Delete Travel Information: Users can select a record and delete it from the app. A confirmation prompt is displayed before deletion.

- Dark Mode Theme: The app features a visually appealing dark mode theme with a black and silver color scheme.

## Requirements

- Python 3.x
- Tkinter module
- csv module

## Usage

1. Install the required dependencies by running the following command:
```
pip install tkinter
```
2. Clone the repository or download the source code.

3. Run the `app.py` file:
```
python app.py
```
4. The Event Management App window will appear, where you can add, search, and delete travel information.
